What this playbook does

This is your personal, private job-search AI system — everything lives in this browser tab's local storage, nothing is sent anywhere. Instead of rewriting prompts from scratch every time, you keep reusable workflows (editable templates), a running Career Facts profile so you never forget a talking point or a gap, and two builders for assembling custom prompts.

  • Career Facts — your durable wins, gaps, transferable skills and domain framing. Workflows can pull straight from this so every prompt stays consistent.
  • Workflow Library — ready-made, editable prompt templates organized by job-search category (LinkedIn, outreach, cover letters, interviews, JD analysis, negotiation).
  • Prompt Builder — assemble a custom prompt from labeled building blocks (role, objective, constraints, etc.) with a live preview.
  • Loop Builder — turn any prompt into a self-checking, iterative "loop" prompt with a goal, evaluation criteria, and stop conditions.

          A quick tour of the Career AI Playbook

          This is a private, self-contained tool for running your job search with AI — everything is saved only in this browser's local storage, nothing leaves this file.

          • Dashboard — your overview: stats, categories, and favorited workflows.
          • Career Facts — a running list of your wins, honest gaps, transferable skills, and domain-framing notes. Fill this in once; workflows can pull from it so you're always consistent.
          • Workflow Library — ready-made, editable prompt templates for LinkedIn, outreach, cover letters, interview prep, JD analysis, and negotiation. Fill in the blanks, copy, done.
          • Prompt Builder — for anything the library doesn't cover. Add labeled blocks (role, objective, constraints…) and watch the prompt build itself.
          • Loop Builder — wraps any prompt in a self-checking iteration loop: draft, score, revise, stop when your conditions are met.

          Use the search bar or press / to find a workflow fast, or ? anytime to see all keyboard shortcuts.
